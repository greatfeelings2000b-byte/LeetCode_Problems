# LeetCode Solution - Ai/Ml Journey 
This repository documents my journey into Artificial Intelligence and Machine Learning by strengthening my Python problem-solving skills. Each solution helps me build a solid foundation in algorithms and data structures, which is essential for Ai/Ml development.
The repo contains solutions to LeetCode problems organized by difficulty: Easy, Medium, and Hard. Problems are further grouped by algorithmic patterns such as arrays, hashing, twp pointers, sliding windows, and hash maps.
My goal is to progress from beginner to intermediate problem-solving, and eventually apply these skills to real AI/Ml projects. This repo serves both as a personal learning archive and as a resource for others preparing for coding interviews or exploring Ai/Ml readiness.
